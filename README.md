# Food Business Landing Page

This project is a landing page designed for a small homemade food business that sells empanadas.

The goal of the page is to present the products in a visually appealing way and make it easy for customers to place orders through WhatsApp.

## Features

- Responsive layout
- Product sections with pricing
- WhatsApp order integration
- Carousel for product images
- Mobile navigation menu
- Smooth scrolling navigation

## Technologies

- HTML5
- CSS3
- Bootstrap
- JavaScript

## What I learned

While building this project I practiced:

- Creating responsive layouts with Flexbox
- Structuring large landing pages with semantic HTML
- Using CSS variables and clamp() for scalable typography
- Implementing a mobile navigation menu
- Integrating external services like WhatsApp ordering links
